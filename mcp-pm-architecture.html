mcp-pm Architecture

Package Manager for the Model Context Protocol Ecosystem

User Interfaces

  • • CLI — 12 subcommands via click + rich
  • • Web Dashboard — FastAPI + HTMX + Tailwind
  • • VS Code Extension — Coming Soon

Core Features

Tech Stack

  • • click (CLI) / rich (Terminal UI)
  • • HTMX + Tailwind CSS (Web UI)
  • • httpx (Async HTTP Client)
  • • Docker Sandbox Integration
